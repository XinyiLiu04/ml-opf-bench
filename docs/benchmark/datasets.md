## Datasets preparation
This ML-OPF benchmark test uses standard datasets in the field of power systems, covering grid topologies of different scales:



| dataset       | Bus | Branch | Generator | Load       |

| IEEE 30-bus  | 30     | 41     | 6        |    | 
| IEEE 118-bus | 118    | 186    | 54       |    |
| IEEE 300-bus | 300    | 411    | 69       |  | 

## Dataset format
All datasets are stored in CSV format and contain core fields such as node parameters, branch parameters, and constraints, which can be obtained on the "Code" page.
