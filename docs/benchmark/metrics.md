# 评价指标
本基准测试从「精度、效率、鲁棒性」三个维度评价 ML-OPF 方法的性能：

## 1. 精度指标
- 电压偏差：$\Delta V = \frac{1}{N} \sum_{i=1}^N |V_i - V_i^*| / V_i^* \times 100\%$（$V_i$ 为预测值，$V_i^*$ 为真值）
- 网损误差：$\Delta P_{loss} = |P_{loss} - P_{loss}^*| / P_{loss}^* \times 100\%$
- 约束满足率：满足电压/功率/潮流约束的算例占比（%）

## 2. 效率指标
- 计算时间：单次 OPF 求解的平均耗时（ms）
- 显存占用：模型推理时的最大显存消耗（MB）

## 3. 鲁棒性指标
- 负荷波动适应率：负荷±20%波动时，约束满足率的保持程度
- 收敛率：连续 100 次求解中收敛的次数占比
